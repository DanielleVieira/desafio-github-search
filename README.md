# desafio-github-search
Criando um App Android para compartilhar seu portfolio de projeto 

* Aplicação para consumo da API do GitHub
* Integração com API REST usando Retrofit
* Persistência de dados com SharedPreferences
* Aplicação de boas práticas de arquitetura e organização de código
